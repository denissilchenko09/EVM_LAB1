# EVM_LAB1
Данная программа генерирует учебные варианты для демонстрации разницы в точности представления вещественных чисел в форматах `float` (32 бита), `double` (64 бита) и `__float128` (128 бит, quad precision) стандарта IEEE 754.

Для каждого случайно сгенерированного числа (с большим количеством знаков после запятой) программа вычисляет его представление в трёх типах, выводит битовое представление (знак, порядок, мантисса) и абсолютную погрешность по сравнению с исходной строкой.

## Возможности

- Генерация заданного количества вариантов, в каждом варианте — заданное количество чисел.
- Для каждого числа сохраняется:
  - Исходное число (в виде строки с 100 знаками после запятой).
  - Битовое представление IEEE 754 для `float`, `double` и `__float128`.
  - Абсолютная погрешность (разница между исходной строкой и округлённым значением, представленным в виде строки).
- Создаются файлы:
  - `variant_1.txt`, `variant_2.txt`, … — только исходные числа (для выдачи ученикам).
  - `answers.txt` — подробная таблица с ответами для преподавателя.

## Требования

- Компилятор GCC (поддержка `__float128` и библиотеки `quadmath`).
- Библиотека `quadmath` (обычно устанавливается вместе с GCC или отдельно, например, `libquadmath0`).
